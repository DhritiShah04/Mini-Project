#### Create a venv and activate it as:
- `python3 -m venv venv`
- `source venv/bin/activate`

#### Install required libraries as:
- `cd backend`
- `pip install -r requirements.txt`

#### Start the backend as:
- `python backend.py`

### Start the frontend as:
In a new terminal: 
- `cd frontend`          
- `npm install`
- `npm start`
